  • Both programs want only one copy of themselves to run
  • Enforced via a global mutex on the GUID of their .NET assembly
  • Implementation copied from Stack Overflow containing the subtle bug of typing the assembly before getting its GUID
  • Both programs therefore think the GUID of the assembly type mutexed by the other program means they are already running
